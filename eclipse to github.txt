To Add the Project from Eclipse to Github
------------------------------------------------
1)Create a Github account and SignIn
2)Start a Project = Create a Github Repository
3)Start Eclipse
4)Goto Perspective -> Git Respositories and Click on add Git Repositors
5)Create a Project in Eclipse
6)Do a right click on Project >Team>Share>Add to git Repo
7)Commit and push the Project to the Repo
8)Commit and push the Every change to the Repo
