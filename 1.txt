linux commands
--------------

The pwd command              display the location of the current working directory.
The mkdir command            create a new directory under any directory.
The rmdir command            delete a directory.
The ls command               display a list of content of a directory.
The cd command               change the current directory.
The touch command            create empty files. We can create multiple empty files by executing it once.
The cat command              create a file, display content of the file, copy the content of one file to another file
The rm command               remove a file
The cp command               copy a file or directory.
The mv command               move a file or a directory form one location to another location.
The rename command           rename files. It is useful for renaming a large group of files.
The head command             display the content of a file. It displays the first 10 lines of a file.
The tail command             displays the last ten lines of the file content. It is useful for reading the error message.
The tac command              reverse of cat command, as its name specified. It displays the file content in reverse order
The more command             display the file content in the same way that the cat command does.
The less command             includes some extra features such as 'adjustment in width and height of the terminal.
The su command               provides administrative access to another user. In other words, it allows access of the Linux shell to another user.
The id command               display the user ID (UID) and group ID (GID).
The useradd command          add or remove a user on a Linux server.
The passwd command           create and change the password for a user.
The cut command (cut -d(delimiter) -f(columnNumber) <fileName> ) select a specific column of a file. The '-d' option is used as a delimiter, and it can be a space (' '), a slash (/), a hyphen (-), or anything else. And, the '-f' option is used to specify a column number.
